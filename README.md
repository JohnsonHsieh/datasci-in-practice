# 数据专案在实务上的各种坑

## 讲题摘要：
这场演讲将讲述团队在从事数据分析专案时与大型企业交手的经验，分享如何在地雷区完成任务的心得。特别适合想要组队接外包数据分析案的团队以及不知到如何启动一个数据专案的企业。

## 讲者简介：
姓名：谢宗震
单位：DSP智库驱动
职位：首席数据科学家
清华统计博士，致力于资料科学顾问服务，跨足零售、製造、新闻、电信、消防、家防…等领域，辅导政府机关、企业、非营利组织导入资料科学。Data for Social Good（D4SG）计画共同发起人，打造一个「资料力，做公益」的交流与媒合平台。
