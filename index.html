资料的极限 - columns

  • 一个问题是否能透过资料解决和资料搜集与记录的方式有关。

资料的极限 - row

  • 样本中的稀有事件或是未曾发生的事件难以配适模型。

资料的极限 - sample size

  • 统计显著性会受到样本数不足所影响。

资料的品质 - missing value

  • 即使在资讯系统中事先规划好栏位，还是有可能失败。

资料的品质 - error log

  • 当感测器报错时，不一定能够用遗漏值插补来处理。

资料的品质 - illogic

A/B医院相互转诊时间差异过大

  • 合乎资料系统逻辑，但是与真实经验不合。

资料的品质 - unclear

  • 预先规划的BI介面，不一定能够正式实行。

资料的品质 - summary

  • 刚拿到一批资料，绝对不能一头栽进去做深入的分析，或是建立机器学习模型。
  • 合格的数据科学家，要先做探索性分析 (EDA) 计算简单统计量与视觉化观察资料。
  • 这些工具不只能描述资料的特性，更重要的是要检验资料的干净度与正确性。
  • 完成资料品质检验的具体方式就是完成一个资料仪表板 (Dashboard)

资料的品质 - dashboard

  • 资料仪表板 (dashboard) 是完成资料品质检验的具体成果。

建立资料仪表板的意义


确认资料已经可以正确的被取出并视觉化呈现，


让组织中的不同团队共享资料，提升全体素养，


减少资料科学团队产生报表的需求，减少内耗，


建立起信任感，初步展现资料科学团队的价值。
